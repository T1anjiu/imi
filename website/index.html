Watch an Egg Movie Search - All-network Movie Resource Search Platform

Project Introduction: The earliest intention of building a website was for personal convenience! When the traffic increased over time, it was necessary to consider upgrading the configuration or refactoring the project. A few days ago, I saw imiphp on Git and decided to try it out. I simply refactored all the pages and introduced the TP's think-template template engine. The entire refactoring took less than a day, so imiphp is indeed very easy to get started with! Keep it up!

Hupu - Data Migration Service for Billions of Data

Project Introduction: As the data scale grows, MySQL is no longer suitable for multidimensional queries of big data, and search engines like ES are needed for multidimensional word segmentation queries. At the current stage, MySQL uses daily partitioned storage and cannot meet long-term queries across days.

How to complete data migration as quickly as possible and migrate data from the database to ES is an important technical point that needs to be evaluated.

In a high IO-intensive scenario, a single request takes 80 milliseconds. imi uses Swoole coroutines to switch between user space and kernel space, fully utilizing the computer CPU, thereby completing massive data migration quickly.

According to Prometheus monitoring statistics, on two 2C 4G machines, imi synchronizes at a rate of 1000 to 1500 records per second, completing data migration of billions of records.

Blog Post: https://blog.csdn.net/qq_32783703/article/details/113576741

Teacher API - Provides Free Interface Call Platform

Project Introduction: Teacher API provides free API data interface call service platform - we are committed to providing users with stable and fast free API data interface services.

Comment:

Before, the server configuration was 8H8G 30M, with a daily traffic of over 3 million requests. Once, a certain interface would cause the server to crash directly due to an error. I accidentally found the imi project of the group owner (Yurun) when searching, so I stayed up late to replace the core code of the program with imi. Fortunately, I had a server with a configuration of 1H2G 5M, and after testing, with Redis, there were no problems with 2 to 3 million requests. Finally, I would like to thank Yurun for his open source project.
